
# CTIA (Certified Threat Intelligence Analyst)

## Overview
**CTIA** is a globally recognized certification from EC-Council focused on **Cyber Threat Intelligence (CTI)**. It teaches security professionals how to collect, process, analyze, and operationalize threat intelligence to proactively identify and mitigate cyber threats.

---

## Exam Information

| Item | Details |
|--------|---------|
| Exam Code | 312-38 |
| Questions | 50 |
| Duration | 2 Hours |
| Question Type | Multiple Choice |
| Passing Score | Varies (typically ~70%) |
